# Steps to reproduce:

requirement: Docker

### Start postgres

```shell
docker run --name repro-test -d -p 5432:5432 -e POSTGRES_PASSWORD=postgres postgres:14-alpine
```

### Create minimal schema

```shell
docker exec -it repro-test psql postgres postgres
```

```sql
CREATE DATABASE test;
\c test
CREATE TABLE test (
    id bigint PRIMARY KEY ,
    value varchar
);
quit;
```

```shell
./gradlew build
```
